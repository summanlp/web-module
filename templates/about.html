About Summa

Summa is written in Python and can be installed using pip. It uses an improvement of the TextRank algorithm introduced by Rada Mihalcea and Paul Tarau on their 2004 paper: TextRank: Bringing Order Into Texts.

Summa is open source software released under The MIT License, which means you can use it, share it and improve it. Its entire source code, including this website, is hosted on our github page.
